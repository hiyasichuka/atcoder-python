# atcoder-python

# 前提条件

Docker Dektopをインストールしていること

# 利用方法

1.下記コマンドで本リポジトリをローカルにクローンする

```shell
git clone git@github.com:hiyasichuka/atcoder-python.git
```

2.VSCodeでディレクトリを開く

```shell
cd atcoder-python
code .
```

3.左下の><マークを押下して、Repopen in Containerをクリックする

# AtCoder参戦手順

1. AtCoderにログインする

```shell
acc login
# AtCoderのログインパスワード認証を行う
```
